# 基础 Skill 模板

适用于简单功能的 Skill 模板。

## 文件结构

```
your-skill/
├── SKILL.md        # 技能描述（必需）
└── index.js        # 可选：如果需要执行代码
```

## SKILL.md 模板

```markdown
# Skill Name

Brief description of what this skill does.

## Triggers

- When user asks about X
- When user mentions Y
- Keywords: keyword1, keyword2

## Behavior

1. Step one
2. Step two
3. Step three

## Notes

Any additional context or constraints.
```

## 示例

参见 `weather` 或 `healthcheck` 技能作为参考。
