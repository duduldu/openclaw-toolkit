# 定时任务 Skill 模板

用于创建 cron 定时任务的技能。

## 使用场景

- 每日/每周定期提醒
- 数据同步任务
- 定期检查和报告
- 清理和维护任务

## 配置示例

```json
{
  "name": "daily-reminder",
  "schedule": {
    "kind": "cron",
    "expr": "0 9 * * 1-5",
    "tz": "Asia/Shanghai"
  },
  "payload": {
    "kind": "systemEvent",
    "text": "每日工作提醒：检查邮件、日历、待办事项"
  },
  "sessionTarget": "main",
  "enabled": true
}
```

## 常见 Cron 表达式

| 描述 | 表达式 |
|------|--------|
| 每小时 | `0 * * * *` |
| 每天 9 点 | `0 9 * * *` |
| 工作日 9 点 | `0 9 * * 1-5` |
| 每周一 9 点 | `0 9 * * 1` |
| 每月 1 号 | `0 0 1 * *` |
| 每 5 分钟 | `*/5 * * * *` |

## 注意事项

1. **时区**：明确指定时区，避免 UTC/本地时间混淆
2. **幂等性**：确保任务可重复执行，不会产生副作用
3. **错误处理**：任务失败时应有日志或通知
4. **资源限制**：避免长时间运行的任务阻塞系统

## 调试技巧

```bash
# 手动触发 cron 任务测试
openclaw cron run <job-id>

# 查看任务执行历史
openclaw cron runs <job-id>
```
